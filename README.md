# Dr. Alexandra Chen - Design Thinking Professor Portfolio

A modern, professional academic portfolio website showcasing research, publications, mentorship impact, and innovation leadership in Design Thinking.

## Overview

This portfolio presents Dr. Alexandra Chen, Director of the Design Thinking Lab at Stanford Graduate School of Business, as a thought leader in human-centered innovation and organizational transformation.

## Key Sections

- **Hero**: Compelling introduction positioning the professor as a design thinking pioneer
- **Research Initiatives**: Showcase of major research projects and their real-world impact
- **Academic Credentials**: Educational background and qualifications
- **Areas of Expertise**: Research domains, methodologies, and industry applications
- **Impact & Leadership**: Mentorship achievements, startup success stories, and industry collaborations
- **Philosophy**: Core teaching and research approach (Empathy First, Collaborative Ideation, Rapid Experimentation)
- **Contact**: Speaking engagements, consulting, and collaboration opportunities

## Technology Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: Custom components with Canvas effects and interactive elements
- **3D Graphics**: Three.js for visual effects

## Features

- Responsive design optimized for all devices
- Smooth animations and transitions
- Interactive research showcase
- Professional credibility-focused messaging
- Modern, clean aesthetic with gradient accents and clean UI

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Customization

All content is centralized in `data/index.ts` for easy updates:
- Navigation items
- Grid showcase items
- Research projects
- Work experience/impact
- Social media links

## Deployment

Deploy to Vercel with a single command:
```bash
npm run deploy
```
